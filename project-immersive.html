02 — Independent Work

Immersive Tablescape
Installation

Year

2024

Type

Independent Work

Role

Concept · Styling · Installation Planning

Context

Personal Practice

Concept

A structured yet intimate dining environment built around material layering, candlelight rhythm, and tactile detail — designed from the moment of arrival through to seated discovery of each element.

immersive-detail-1.jpg — 800×600px
immersive-detail-2.jpg — 800×600px

Process

  • Designed table layout from guest perspective (arrival view → seated view → detail discovery).
  • Structured layering system: base linen → charger → plate → glassware → vertical accents.
  • Considered comfort, reach, spacing, and safety within the composition.
  • Created setup order and packing checklist for efficient on-site execution.
  • Balanced symmetry with intentional asymmetry for visual interest.

Skills Demonstrated

Spatial Awareness Guest Experience Logic Styling Systems Installation Planning Material Layering
